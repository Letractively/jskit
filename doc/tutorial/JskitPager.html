JskitPager.js
Author:
AnyRock
Initializtion:
var jp = new JskitPager("jp");
Required:
base/*
Methods:
Name Parameters Return Comment
setCssNormal String 翻页链接的缺省样式
setCssSelected String 翻页链接的当前页样式
setCssDisabled String 翻页链接无动作样式
setCssFpnlText String 翻页Fpnl样式翻页器的文本样式
setPageNo int 当前页码
setPageSize int 每页显示的记录数
setTotalSize int 记录总条数
setStyle string 显示的风格，fpnl:首页上一页下一页末页，number:数字页码（缺省）
setFpnlText Array fpnl风格的文字内容,缺省：new Array("First","Prev","Next","Last")
setAction string 翻页动作,可以是链接，或者javascript:开始的脚本，其中$1作为可替换的Parameters，替换为页码。
outContent([rPageNo]) rPageNo: 当前页码 String 获得翻页的HTML代码
display([rPageNo]) rPageNo: 当前页码 在页面打印翻页的HTML代码
