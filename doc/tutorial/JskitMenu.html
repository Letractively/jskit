JskitMenu.js
构造多级菜单
Author:
AnyRock
Initializtion:
var JskitMenu = new JskitMenu("JskitMenu");
Required:
base/*
Methods:
Name Parameters Return Comment
load(rXmlUrl) (String)rXmlUrl: 菜单Configuation:文件(xml格式)的url地址 导入菜单数据
loadXml(rXmlContent) (String)rXmlContent: 菜单Configuation:数据(xml格式) 导入菜单数据
expand() 展开菜单
add(rParent, rCaption, rUrl, rTarget, rCss, rDirection, rLocation) (XmlElement)rParent: 父节点，如果为null则为根节点
(String)rCaption：菜单项标题
(String)rUrl: 菜单项链接
(String)rTarget: 链接打开方式
(String)rCss: 菜单项样式
(Int)rDirection: 子菜单展开方式
(Int)rLocation: 子菜单相对于父菜单项显示的位置
添加一个菜单项
"打开方式"Parameters可用值,缺省为_self：
"_self"
"_blank"
"_top"
"_parent"
"_opener"
"子菜单的展开方向"Parameters可用值：
1(jm.Direction.VER) 横向
2(jm.Direction.HOR) 纵向
"子菜单显示的位置"Parameters可用值：
1(jm.Location.UP) 上方
2(jm.Location.RIGHT) 右方
3(jm.Location.DOWN) 下方
4(jm.Location.LEFT) 左方
push(rParent,rNode) (XmlElement)rParent: 父节点，如果为null则为根节点
(XmlElement)rNode: 新的节点 		添加一个菜单项
getContent() String 获取可见菜单的Html代码
getData() XmlDocument 获取菜单数据
remove() 移除当前选中的菜单项
move(rParent) (XmlElement)rParent: 父节点 将当前选中的菜单项移动到指定的父节点下
select(rId) (String)rId: 节点idAttributes:值 选中节点对象
select(rElement) (XmlElement)rElement: 节点 选中节点对象
setDynamic(rDynamic) (boolean)rDynamic: 菜单是否动态显示，缺省为false 设置菜单是否动态显示
setRootDirection(rDirection) (Int)rDirection: 菜单张开方式 设置菜单张开方式
setIsShowArrow(boolean v) 是否显示指示箭头，缺省为ture
setUpArrowHtml(String v) (String)v : html代码 设置上箭头▲的html代码，替换内置的html代码
setRightArrowHtml(String v) (String)v : html代码 设置右箭头►的html代码，替换内置的html代码
setleftArrowHtml(String v) (String)v : html代码 设置左箭头◄的html代码，替换内置的html代码
setDownArrowHtml(String v) (String)v : html代码 设置下箭头▼的html代码，替换内置的html代码
clone(String jskitMenuName) (String)jskitMenuName: 新的jskitMenu对象的名字 JskitMenu 克隆一个新的JskitMenu对象
expandTree() 输出一个树形菜单
setTreePrefixCssClass(String v) (String)v : style class name 设置树形菜单结构线条的样式
setTreeFolderCssClass(String v) (String)v : style class name 设置树形菜单非叶子节点的样式
setTreeLeafCssClass(String v) (String)v : style class name 设置树形菜单叶子节点的样式
setTreeFolderSignCode(String v) (String)v : html code 设置树形菜单非叶子节点标识
setTreeLeafSignCode(String v) (String)v : html code 设置树形菜单叶子节点标识
Comments:
菜单样式的定义规则:
如果某个菜单的样式定义为class1,那么该菜单项鼠标悬停的状态为class1_over,被选中的状态为class1_selected
