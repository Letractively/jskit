Required:

Methods:
Name Parameters Return Comment
setTimeout(Integer time) time : 时间 设定任务执行的等待时间，单位为秒；如果某任务执行时间超过设定的过期时间，则执行下一个任务。缺省：5秒
addUrlTask(String taskName,String url) taskName : 任务名称
url : 执行任务的url 		增加一个url类型的任务，该任务提交给一个内嵌的iframe对象
addFuncTask(String taskName,String funcName) taskName : 任务名称
funcName : 方法名 		增加一个函数型的任务，该任务执行指定的方法名
done() 通知当前任务状态为完成
setLogOuter(HtmlElelemt obj) obj : html对象 指定日志输出对象
start() 开始执行
pause() 暂停执行
stop 所有任务终止
load(Array tasks) tasks : 任务列表,json格式
