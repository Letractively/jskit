JskitContextMenu.js
页面的右键菜单
Author:
AnyRock
Initializtion:
var jskitContextMenu = new JskitContextMenu();
Required:
core/*
Configuation:
Name DataType DefaultValue Comment
configuation.open boolean 无 是否使用自定义右键菜单
configuation.provider String jskitToolbar.getContent 指定菜单内容的来源，由哪个函数来提供，该函数至多有一个eventParameters，传递右键点击的事件
Example:
//在JskitConfig.js中Configuation:JskitContextMenu
//下面例子中使用了jskitToolbar.getContentMethods:，如使用此Methods:请确定"page/JskitToolbar.js"被装载
jskitContextMenu.configuation.provider = "jskitToolbar.getContent";
//下面例子中使用了自定义的Methods:
jskitContextMenu.configuation.provider = "myContextMenu";
//myContextMenu的定义如下：
<script>
function myContextMenu(rEvent){!!注意：Methods:至多有一个Parameters，且只能是event!!
    return "menu item";
}
</script>
