JskitImport.js
Author:
AnyRock
Comment:
用来引入所需要的js和css文件
Initializtion:
static $import
Required:
nothing
Methods:
Name Parameters Return Comment
path([rKey]) rKey : 判断位置的参考正则表达式，缺省为"/jskit\.core([0-9|a-z|A-Z|\.]*)\.js"，即为得到Jskit.org的根路径 String 根据参考rKey值得到Url path
js(rFileUrl[,rKey]) rFileUrl : js文件的路径
rKey : 参考文件名 		如果该文件已经被JskitLoad装载过,不会被再次装载
css(rFileUrl[,rKey]) rFileUrl:css文件的路径
rKey : 参考文件名 		如果该文件已经被JskitLoad装载过,不会被再次装载
Example:
$import.js("ui/JskitValidation.js");
$import.css("JskitXMenu.css","/JskitXMenu.js");
