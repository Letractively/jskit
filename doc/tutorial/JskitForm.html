  • JskitForm.SELECT
  • JskitForm.CHECKBOX

JskitForm.SELECT
初始化
var jf = new JskitForm("jf");
var _select = new jf.SELECT("#selectId");
Methods:
Name Parameters Return Comment
isNaN() boolean 当前对象是否为null
selectByValue(value) value: option value 根据value设置对应的option被选中，如果有重复，第一个有效
selectByIndex(rIndex) rIndex:option的索引 String 根据option的索引,设定其选中
getValue() String 获取当前对象的value
getValueByIndex(rIndex) rIndex:option的索引 String 根据option的索引,得到其value
setValueByIndex(rValue,rIndex) rValue: option的value
rIndex: option的索引 		根据option的索引,设置其value
getValueByText(rText) rText: option的text 根据option的text,得到其value,如果text有重复,取第一个
setValueByText(rValue,rText) rValue: option的value
rText: option的text 		根据option的text,得到其value,如果text有重复,取第一个
add(rValue,rText[,rIndex]) rValue: option的value
rText: option的text
rIndex: option的index 		添加一个option对象,如果rIndex未设置,则添加在options的末尾
push(rOption[,rIndex]) rOption: option对象
rIndex: option的index 		添加一个option对象,如果rIndex未设置,则添加在options的末尾
remove(rArg) rArg:
如果rArg是一个字符串,则视为option的value
如果rArg是一个整数,则视为option的index 		根据option的value或index,移除该option对象
removeAll() 移除所有option对象
JskitForm.CHECKBOX
初始化
var jf = new JskitForm("jf");
var _cb = new jf.CHECKBOX("#cbId");
Methods:
Name Parameters Return Comment
isNaN() boolean 判断当前对象是否为null
getValue() string 获取当前对象的value,形如"v1,v2,v3..."
setValue(v) v: value值 给据value设置当前对象的选中状态
