Methods:
Name Parameters Return Comment
addColumn(String rKey,String rTitle[,rHidden]) rKey: 字段Name(不能重复)
rTitle: 数据表显示的列Name
hidden: 是否为隐藏Attributes:，隐藏的列，在out时不被输出 		增加一个列,
如：addColumn("sex","性别");
setColumn([column1][,column2]...) column1...列名，不能重复 同时设置所有列，列的title同key
insert([data1][,data2][,data3]....) rData: 一条记录 按列顺序，填入对应的数据
如：insert(1,"Edwon","Male");
getTable([beginIndex,count]) beginIndex: 起始行索引
count: 行数 		html代码 得到table的HTML代码，out(1,3)表示从索引1的行开始，打印3行
getBodyContent([beginIndex,count]) beginIndex: 起始行索引
count: 行数 		html代码 得到table中tbody部分(包括thead)的HTML代码
getHeadContent() html代码 得到table中thead部分(包括thead)的HTML代码
out([beginIndex,count]) beginIndex: 起始行索引
count: 行数 		打印getTable()取得的内容
getValue(rRowIndex,rCellIndex) rRowIndex: 行索引
rCellIndex: 列索引或者列名 		string 得到某行某列的值（单元格内容）
setValue(rRowIndex,rCellIndex,rValue) rRowIndex: 行索引
rCellIndex: 列索引或者列名
rValue: 值或内容 		设置某行某列的值（单元格内容）
clear() 清楚table中的数据
setTableCssClass(cssClass) cssClass: 样式定义 设置table的样式定义
setTHeadCssClass(cssClass) cssClass: 样式定义 设置thead的样式定义
setTBodyCssClass(cssClass) cssClass: 样式定义 设置tbody的样式定义
setColumnCssClass(rCIdx,cssClass) rCIdx: 列的key值或索引值
cssClass: 样式定义 		设置某一列单元格的样式定义,rCIdx可以是索引值（整数），也可以是列的键值（字符串）。
setAction(actionName,functionName) actionName: 事件Name
functionName: 响应时间的Methods:Name（注：没有括号部分） 		设置单元格的事件，如setAction("onclick","showMe")
1)Methods:只提供Methods:名，该Methods:有且只有一个Parameters，即该单元格对象。如:showMe(obj){//do sth.}
2)可通过单元格Attributes:获取行索引和列索引，如obj.getAttribute("row")得到行索引，obj.getAttribute("col")得到列索引。
