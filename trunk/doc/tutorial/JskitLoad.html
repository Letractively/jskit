JskitLoad.js
Author:
AnyRock
Comment:
JskitLoad将自动装载所需要的jskit文件，其中已经定义必须的基础文件"base/*"是缺省装载的，如有特定需求要装载更多的js文件，请修改JskitLoad.js文件，将需要添加的Js文件写在脚本最后边。
Initializtion:
var jskitLoad = new JskitLoad();
Required:
base/*
Methods:
Name Parameters Return Comment
path([rKey]) rKey : 判断位置的参考文件名，缺省为"/JskitLoad.js"，即为得到Jskit.org的根路径 String 根据参考rKey值得到Url path
loadScript(rFileUrl[,rKey]) rFileUrl : js文件的路径
rKey : 参考文件名 		如果该文件已经被JskitLoad装载过,不会被再次装载
loadCss(rFileUrl[,rKey]) rFileUrl:css文件的路径
rKey : 参考文件名 		如果该文件已经被JskitLoad装载过,不会被再次装载
Example:
jskitLoad.loadScript("model/JskitValidation.js");
jskitLoad.loadScript("JskitXMenu.css","/JskitXMenu.js");
