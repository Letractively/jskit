JskitAnimation.js
对象移动
Author:
AnyRock
Initializtion:
var ani = new JskitAnimation("ani");
Required:
base/*
Methods:
Name Parameters Return Comment
moveTo(HTMLElement rObj,Float rX,Float rY[,Int rSpeed][,String rOnFinish]) rObj : 移动的对象
rx : 目标x坐标
ry : 目标y坐标
rSpeed : 移动速度(10-100之间的整数,数值越大速度越快,缺省为50)
rOnFinish : 移动结束之后触发的Methods: 		将对象移动到某一个位置
move(HTMLElement rObj,Array rPath[,Int rSpeed][,String rOnFinish]) rObj : 移动的对象
rPath : 移动的轨迹,由一组x,y坐标数组组成
rSpeed : 移动速度(大于1的整数,数值越大速度越快,缺省为10)
rOnFinish : 移动结束之后触发的Methods: 		将对象移动到某一个位置
transform(HTMLElement rObj,Float rX,Float rY,Float rWidth,Float rHeight[,Int rSpeed][,String rOnFinish]) rObj : 移动的对象
rx : 目标x坐标
ry : 目标y坐标
rWhdth : 目标宽度
rHeight: 目标高度
rSpeed : 移动速度(10-100之间的整数,数值越大速度越快,缺省为50)
rOnFinish : 移动结束之后触发的Methods: 		将对象移动到某一个位置的同时改变大小
Example:
查看效果演示页面

moveTo():,将一个对象移动到指定的位置

move():,将一个对象按一定的轨迹移动

transform():,将一个对象变形并移动到指定位置
