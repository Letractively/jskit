JskitRichDropDownList.js
异步模式下拉列表，可实现级联
Author:
AnyRock
Initializtion:
Required:
base/*.*
server/JskitXmlHttp.js
Methods:
Name Parameters Return Comment
deploy(rTextFeildID,rValueFeildID,rUrl,rParmName,rParmFeildID) rTextFeildID: 显示Text的表单项ID
rValueFeildID: 保存Value值的表单项ID
rUrl: 获取下拉列表数据的webservice的URL
rParmName: 参数名称
rParmFeildID: 保存参数值的表单项ID
必须在页面装载完成后使用。
通过ajax方式获取的数据必须是json格式的。
如果设置了rParmName和rParmFeildID，会在Url后面自动增加&ParmName={ParmFeild.value}，方便webservice获取数据时进行过滤。
deployTree(rTextFeildID,rValueFeildID,rUrl,rParmName,rParmFeildID) rTextFeildID: 显示Text的表单项ID
rValueFeildID: 保存Value值的表单项ID
rUrl: 获取下拉列表数据的webservice的URL
rParmName: 参数名称
rParmFeildID: 保存参数值的表单项ID 		按树形结构显示所有分类项，且只有叶子节点可以选择。适合分类数据少，分类层级不固定的数据。
deployStatic(rTextFeildID,rValueFeildID,rData) rTextFeildID: 显示Text的表单项ID
rValueFeildID: 保存Value值的表单项ID
rData: 下拉列表数据 		不通过ajax方式加载数据，直接取用通过参数传递进来的已有的数据
setSelectAction(rFuncName) rFuncName：方法名（不包括参数），系统自动为其传递选择项的Key和Value两个参数
如：setSelectAction("aaa"),实际执行aaa(key,value); 		设置选中下拉选项之后触发的动作
setDebugMode(rMode) rMode：模式值(0,1;yese,no;true,false) 设置调试模式，如果是调试模式，返回错误信息时可以点击错误信息，弹出ajax请求页面。
setAutoWidth(rVal) rVal：值(0,1;yese,no;true,false) 设置是否根据输入框的大小自动调节下拉列表的宽度。
setWidth(rVal) rVal：数值 设置下拉列表的宽度。
getKey(rIndex) rIndex：下拉项的索引值 根据下拉项的索引，获得该项的Key值
getValue(rIndex) rIndex：下拉项的索引值 根据下拉项的索引，获得该项的Value值
setEmptySelectText(rString) rString：空下拉项显示的内容 设置空下拉项显示的内容
setIsMultiSelect(rBoolean) rBoolean：是否多选,0|1,yes|no,true|false 设置是否多选
