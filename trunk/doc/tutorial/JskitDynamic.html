Methods:
Name Parameters Return Comment
setMovableTags(rTags) rTags: 允许被拖动的对象的TagName，多个用逗号分隔，不区分大小写，缺省为：“div,font,span,table,a,img” 设定哪些对象（标签）允许被拖动，例如:setMovableTags("div,table"),只有div和table可以被拖动。
setUnmovableTags(rTags) rTags: 不允许被拖动的对象的TagName，多个用逗号分隔，不区分大小写，缺省为：“body,head,html,script,style,form,meta,frame,textarea,input,frameset,link” 设定哪些对象（标签）不允许被拖动，例如:setUnmovableTags("div,table")，则div,table不能被拖动，该优先级高于setMovableTags。
setBlockTags(rTags) rTags: 不触发拖动事件的对象的TagName，多个用逗号分隔，不区分大小写，缺省为：“textarea,input,select,button,a” 当鼠标事件发生在这些标签上时，不触发拖动事件。比如一个可以移动的div中有一个textarea对象，那么拖拽该textarea时不会触发div的移动。
setRealTime(rRealTime) (boolean)rRealTime :是否是实时的，缺省为false 如果设置为实时，移动时对象也一起移动。否则，先移动cover，移动结束后对象位置切换到移动后的坐标。
getRealTime() boolean 获得realtimeAttributes:值
setOnMoveDone(rFunctionString) (string)rFunctionString :执行的程序代码 设置在移动结束后执行的程序代码，如：setOnMoveDonw("aletrt(1)");
stop() 停止对当前对象的拖动控制
add(xpath) xpath: 查找对象的xpath 将需要移动的对象注册到jskitDynamic的队列中
remove(xpath) xpath: 查找对象的xpath 将需要移动的对象从jskitDynamic的队列中移除
pause() 暂停jskitDynamic的功能
play() 从暂停恢复jskitDynamic的功能
getOffsetX() integer 获得移动的x偏移量(像素)
getOffsetY() integer 获得移动的y偏移量(像素)
