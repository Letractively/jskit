$$(Object).effectStyle(rStyleName); 得到当前的css样式值 string obj.effectStyle("color"),得到当前对象obj的前景色值，兼容IE和FireFox

String
Comment:
该内容定义在base/JskitBase.js中
扩展string的Methods:
Name Parameters Return Comment
String.trim() 去除首位空格 string "aaa".trim()
String.encodeHtml() 编码一段html代码 string "<font>aaa</font>".encodeHtml();
将的得到:
&lt;font&gt;aaa&lt;/font&gt;
String.decodeHtml() 解码一段html代码 string "&lt;font&gt;aaa&lt;/font&gt;".decodeHtml();
将的得到:
<font>aaa</font>
String.toInt() 转换为整型值 int "1".toInt()等同于parseInt("1");
String.toFloat() 转换为浮点型值 float "1.01".toFloat()等同于parseFloat("1.01");
String.toFormatFloat([rDefault][,rLength]) 转换为指定小数个数的浮点型值
rDefault: 如果是非法的值，返回rDefault
rLength: 指定要显示的小数位数，缺省为2 		float "1.013101".toFormatFloat(0,3)返回1.013;
String.cut(int rLen[,string rTail]) rLen: 要截取的长度
rTail: 尾部追加字符串 		string 将长字符串截断
String.select(value1[,value2][,value3]...) value1...: 可选择的值 string 在选定的范围内赋值。
例如在设置form的提交方式时，只有get和post两种，如果代表method值的变量为m,可写成form.method=m.select("get","post")，m从get和post中选择值，如果没有匹配(m不在这些值的范围里)，则Return第一个值(此例中为get)。
如果Parameters错误(Parameters个数少于1或Parameters不是字符串)，Return"";
String.isMatch(String rPattern,String rOptions) rPattern: 正则表达式
rOptions: 参数:g,i 		boolean 测试是否匹配正则表达式
String.replaceAll(String rPattern, String rNewStr,String rOptions) rPattern: 正则表达式
rNewStr: 替换成的字符串
rOptions: 参数:g,i 		string 全局替换
String.clearOffHTML() string 清除字符串中的html标签
