Methods:
Name Parameters Return Comment
setTitle(string rTitle) rTitle:窗体标题 设置窗体标题
setContent(string rHTML) rHTML:窗体中显示的内容 设置窗体居中显示
setStyle(rParams) rParams:样式串形如："width=100,height=200,left=10,top=10" 目前支持的样式属性：
width: 窗体宽度,数值
height: 窗体高度,数值
left: 窗体x坐标,数值
top: 窗体y坐标,数值
scrolling: 是否滚动条,yes|no,true|false
center: 窗体居中显示,yes|no,true|false
class: 窗体样式,css串
titleclass: 标题样式,css串
bodyclass: 内容样式,css串
refresh() 刷新窗体
open() 打开/显示窗体
minimize() 最小化/隐藏窗体
close() 关闭窗体
getId() string 得到窗体ID
getContent() 得到窗体内容
setZIndex() 设置窗体的zindex值
resizeTo() 修改窗体的大小
fullScreen() 全屏显示
