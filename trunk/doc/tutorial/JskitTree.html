JskitTree.js
构造树形菜单
Author:
AnyRock
Initializtion:
var jskitTree = new JskitTree("jskitTree");
Required:
base/*
Methods:
Name Parameters Return Comment
newNode(rParent,rOpen,rText,rUrl[,rTarget]) (Object)rParent: 父对象
(Boolean)rOpen : 缺省状态,true开启,false关闭
(String)rText : 菜单标题
(String)rUrl : 菜单链接,""或"#"为无链接
(Enum)rTarget : _self,_blank 		Object 创建一个新的菜单节点对象
setPath(rPath) (String)rPath: 节点链接的前缀 设置节点链接的前缀，如setPath("doc/"),节点所有的链接都以doc/开始。
setPrefixText(rPrefix) (String)rPrefix: 节点的前缀字符串，缺省为"·" 设置节点的前缀字符串
setClosePrefix(rPrefix) (String)rPrefix: 节点关闭状态的前缀字符串，缺省为"4" 设置节点关闭状态的前缀字符串，在webdings字体下，4显示为4
setOpenPrefix(rPrefix) (String)rPrefix: 节点打开状态的前缀字符串，缺省为"6" 设置节点打开状态的前缀字符串，在webdings字体下，6显示为6
out(rTarget) rTarget可选择类型:
等于"html"的字符串 : Return一段html代码
不等于"html"的字符串 : 输出到id=rTarget的对象的innerHTML
Html 对象 : 输出到该对象的innerHTML
其他 : 直接打印输出 		Object 创建一个新的菜单节点对象
样式表：
菜单样式的定义规则:
关闭状态: "l"+菜单层级+"_skitTreeNodeText"
开启状态: "l"+菜单层级+"_skitTreeNodeText_active"
叶子节点的前缀样式: "JskitTreePrefix"
目录节点的前缀样式: "JskitTreePrefix_action"
