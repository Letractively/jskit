Methods:
Name Parameters Return Comment
addColumn(rKey,rDefaultValue,rEmpty,rDistinct) rKey: 字段Name(区分大小写)
rEmpty:是否允许空,缺省为true
rDistinct: 是否不允许重复，缺省为false(如果允许空，该Parameters被置为false) 		增加一个列,如：
addColumn("name",false,true);//字段名name,不允许为空，不允许重复
addColumn("sex",false);//字段名sex，不允许为空，可以重复
addColumn("birthday");//字段名birthday，允许为空，可以重复
setColumn([column1][,column2][,column3]....) [column1]...字段名 增加三个列,如：
addColumn("name","sex,"birthday);//字段名name,不允许为空，不允许重复 每个列的empty和distinctAttributes:均为缺省
insert([data1][,data2][,data3]....) 按列顺序，填入对应的数据,如：
insert("Edwon","Male","1900");//name,sex,birthday
row(rIndex) rIndex: 行索引 Row() 得到一行
count() integer Return行数
