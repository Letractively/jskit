JskitTaskTimer.js
构造树形菜单
Author:
AnyRock
Initializtion:
var jskitTaskTimer = new JskitTaskTimer("jskitTaskTimer");
Required:
base/*
Configuation:
Name DataType DefaultValue Comment
interval int 1000 任务执行间隔的毫秒数
autoRun boolean true 是否页面装载就开始执行任务
checkStatus boolean true 是否要验证任务执行执行完成(提交的页面已经装载则证明该任务完成,执行下一个任务)
monitorCssClass String 进度文字的样式定义
Methods:
Name Parameters Return Comment
newTask(rUrl,rTitle) (String)rUrl: 要执行的url
(String)rTitle : 任务Name 		增加一个任务
run() 执行任务
